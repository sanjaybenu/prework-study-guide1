HTML

  • HTML stands for Hypertext Markup Language
  • It is used to add elements to the website, like Text, Image, Header, List etc.
  • Elements to divide and describe a page are also added here
  • The head element contains information about the webpage.
  • Head contains metadat elements to help render the page correctly and are not visible to the user
  • The body element represents the visible content shown to the user.
  • Body elements contains the content visible to user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS file should be linked to HTML through link element
  • Multiple selector can be styled using a comma using DRY principles
  • Style can be applied using class attribute to apply it the elements belonging to a class

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds a change to the working directory to the staging area
  • git commit -m "your message": commits changes with message
  • git pull: used to integirate remote changes
  • git push: used to upload local repository changes remotely
